Objective: To analyze customer demographics and spending patterns to identify key target audiences for a retail environment.

📌 Business Problem
The mall management has a list of customers but doesn't know which groups contribute most to their success. I am tasked with analyzing this data to find "High-Value" groups and "Under-Served" groups to help the marketing team stop guessing and start using data.

🎯 Project Objectives
Data Cleaning & Integrity: Process raw customer data using Python/Pandas to ensure it is ready for analysis.

Descriptive Analytics: Use statistical summaries to find the median income and spending scores across different age groups and genders.

Pattern Recognition: Use data visualization (Seaborn/Matplotlib) to identify natural groupings in customer behavior.

Custom Segmentation: Develop a rule-based classification system to categorize customers into actionable segments (e.g., "Budget-Conscious," "Big Spenders," "Window Shoppers").

🛠️ Tools Used
Python: Main language for analysis.

Pandas: Used for data manipulation, grouping, and filtering.

Matplotlib/Seaborn: Used for creating distribution plots and scatter plots to visualize customer trends.

📈 Key Deliverables
A Jupyter Notebook showing the step-by-step process from raw data to insights.

Visualizations that clearly show the relationship between income and spending.

A summary of 3-5 specific "Customer Personas" found during the analysis.
